# Trolling-Pickleball---Python-Project
A ping-pong game with a lot of little twists - taking inspirations from the game of Pong, Trolling Pickleball is an addition of unpredictable twists to make the game more chaotic and fun.

This project was created as a school group project to explore game development using the language of Python.

# How to run


1. Libraries to install:
   
  pip install numpy
  
  pip install scikit-learn
  
  pip install pygame


3. Run the program:
   
  python Trolling_Pickleball.py
